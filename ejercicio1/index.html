Ejercicio 1: Suma de dos números

Escribe un programa en Python que solicite al usuario dos números y luego imprima la suma de ambos.

Enunciado:

Crea un programa que haga lo siguiente:

  1. Solicite al usuario que ingrese el primer número.
  2. Solicite al usuario que ingrese el segundo número.
  3. Calcule la suma de los dos números ingresados.
  4. Imprima el resultado de la suma.

Ejemplo de salida:

Ingresa el primer número: 5
Ingresa el segundo número: 3
La suma de 5 y 3 es 8

Solución en Python:

        
        
# Solicitar el primer número al usuario
numero1 = float(input("Ingresa el primer número: "))

# Solicitar el segundo número al usuario
numero2 = float(input("Ingresa el segundo número: "))

# Calcular la suma de los dos números
suma = numero1 + numero2

# Imprimir el resultado
print(f"La suma de {numero1} y {numero2} es {suma}")
